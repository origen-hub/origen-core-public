# # ADP Overview

Automatic Digital Provenance is the structural result of OriGen’s deterministic, digest‑pinned workflow model.

ADP explains how provenance emerges naturally when:

* toolchains are defined explicitly in Git,
* builds are immutable by digest,
* planning is pure and non‑executing,
* and Routes capture complete workflow intent.
